# Organist

基于 Next.js 16 的现代化网站项目，支持多语言（简体中文、英语）。

## 技术栈

- Next.js 16 (App Router)
- TypeScript
- SCSS（非 module）
- Tailwind CSS
- shadcn/ui
- next-intl (i18n)

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目已配置 GitHub Pages 自动部署，推送到 `main` 分支时会自动构建和部署。

域名：https://kimitinorganist.cn
